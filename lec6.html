[Lecture 6] Variables

In computer programming, a variable is like a small box (located in the RAM) in which you can store data. The amount and the kind of data that a given variable can accommodate depends on the type of the variable. As it turns out we use variables all the time to perform computations just like how we use variables in mathematics and physics. To distinguish among different variables, each variable is given a unique name at the time of creation.
